# Tarea: A* (A-Star), Heurística y Teoría de Juegos en Python

## Descripción

Este repositorio contiene el desarrollo de una tarea académica sobre temas fundamentales de inteligencia artificial y toma de decisiones, implementados en **Python** y trabajados en **Google Colab**.

La actividad incluye:

- Investigación teórica sobre el algoritmo **A\* (A-Star)**
- Explicación del concepto de **heurística**
- Desarrollo de ejemplos sobre **teoría de juegos**
- Implementación de códigos en Python
- Visualización y explicación de resultados

---

## Contenido del proyecto

### 1. A* (A-Star)
Se presenta la parte teórica del algoritmo A\*, su funcionamiento y una implementación en Python que permite:

- Buscar rutas en una cuadrícula
- Evitar obstáculos
- Encontrar el camino más corto hacia la meta
- Mostrar el recorrido encontrado

### 2. Heurística
Se explica el concepto de heurística y su relación con A\*:

- ¿Qué es una heurística?
- ¿Para qué sirve?
- ¿Cómo funciona?
- Ejemplos de heurística:
  - Distancia Manhattan
  - Distancia Euclidiana

También se incluye una comparación práctica en Python para observar cómo distintas heurísticas pueden influir en la exploración de caminos.

### 3. Teoría de Juegos
Se desarrolla la teoría básica de juegos aplicada a decisiones estratégicas entre agentes.

Incluye:

- ¿Qué es la teoría de juegos?
- ¿Para qué sirve?
- ¿Cómo funciona?

Además, se implementan 3 ejemplos interactivos en Python:

1. **Dilema del prisionero**
2. **Juego de coordinación**
3. **Piedra, papel o tijera**
